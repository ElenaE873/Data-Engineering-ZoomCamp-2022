# Week - 2 - Learning || Data Lake

A "repository" that holds Big Data.
Must be secure, scalable and hopefully inexpensive.

### Data Lake vs. Data Warehouse

Data Lake:
- raw
- large
- undefined
- ELT solution

Data Warehouse:
- Refined
- Smaller
- Relational
- ETL solution
